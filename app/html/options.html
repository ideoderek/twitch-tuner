Feedback

Report bugs

Submit bug reports to the GitHub repository by creating a new issue, or providing additional information to an existing issue. Or email twitchtuner.ideoderek@antichef.com.

Request features

Submit feature requests to the GitHub repository by creating a new issue. Or email twitchtuner.ideoderek@antichef.com.

Contribute

Translate

To contribute translations, email a request to twitchtuner.ideoderek@antichef.com for a list of user-visible strings.

Develop

To contribute code, checkout the GitHub repository and submit a pull request.
